# BSS-Project
Sistem Informasi Manajemen Battery Swapping Station berbasis Java Swing dan MySQL.

## Cara Menjalankan:
1. Clone atau Download repository ini.
2. Buka folder di NetBeans.
3. Import file `bs_station.sql` ke database MySQL (XAMPP).
4. Pastikan library MySQL Connector JDBC sudah terpasang di project.
5. Jalankan `Dashboard.java`.
